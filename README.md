## Weather App

A simple, modern **Weather App** built with **Cursor**, using the **Open-Meteo API** to look up a city and fetch the current temperature.

## Features

- Search by city name
- Fetches current temperature from Open-Meteo (no API key required)
- Clean, modern UI with responsive layout
- Friendly error message for invalid cities or network issues

## How to Run

1. Open `c:\Apps\my-first-ai-app\index.html` in your browser.
2. Enter a city name and click **Search**.

Tip: For best results, run it from a simple local server (optional).
